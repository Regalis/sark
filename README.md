SARK
====

Scorpion Apache (Rank) Killer

Apache **Remote** Denial of Service (memory exhaustion)

SARK is a simple tool for testing Apache configuration.

Requirements
============

Just Python 3.x

Usage
=====

	$ ./sark.py [-t NUM] [-i SEC] [-v] TARGET-1-URL TARGET-2-URL (...)

Options
=======

* -t NUM, --threads NUM - the number of threads that will be used to single attack
* -i SEC, --interval SEC - interval betwean targets
* -v, --verbose - enable verbose messages
* -h, --help - print help message and exit
* -u, --usage - print help message and exit
